# BullStation

ダーツのアウト（出上がり）を練習・習得するための学習支援Webアプリです。

## 機能概要

### アウトルール
3種類のルールに対応しています。

| ルール | 上がり条件 |
|--------|-----------|
| オープン | 任意のダーツで上がり可能 |
| ダブル | ダブルまたはブルズアイで上がり |
| マスター | ダブル・トリプル・ブルズアイで上がり |

### 学習モード
ランダムに出題される点数（21〜180）に対して、3本以内のダーツでアウトを組み立てます。
- 正誤判定後に模範解答を最大5件表示
- セッション中の正答率をリアルタイム表示
- 直近10問は重複出題しない

### タイムアタックモード
10問を連続出題し、全問解答までのタイムを計測します。
- 0.1秒単位でタイム表示
- 結果画面で全問の回答履歴を確認
- ハイスコアを最大10件ローカルに保存

### ダーツボード
SVGで描画された対話的なダーツボードからダーツを選択できます。
- シングル・ダブル・トリプル・ブル・ブルズアイに対応
- スコア表示 / 外枠番号表示の切り替え
- 選択中のダーツをハイライト表示

## セットアップ

```bash
yarn install
yarn dev
```

ブラウザで `http://localhost:5173` を開いてください。

## ビルド

```bash
yarn build
```

## 技術スタック

- **React 18** + **TypeScript**
- **React Router DOM 6** — SPA ルーティング
- **Tailwind CSS 3** — スタイリング
- **Vite 4** — ビルドツール / 開発サーバー
- **SVG** — ダーツボード描画
- **LocalStorage** — 設定・スコアの永続化
